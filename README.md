# Heap-sort-algorithm-using-HTML-CSS-JAVASCRIPT-and-Python-flask
 

An interactive full-stack web application that visualizes the Heap Sort algorithm in real time.  
This project is designed to help understand heap operations through dynamic visual feedback using multiple representations.

## âš™ï¸ How It Works

1. Users insert values or generate a random array.
2. The frontend sends the array and selected heap strategy (Min/Max) to the backend.
3. Flask processes the Heap Sort algorithm and records each step.
4. The backend returns step-by-step states via a REST API.
5. The frontend animates each step across:
   - Array view
   - Tree structure
   - Bar visualization
